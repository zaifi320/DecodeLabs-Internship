
# 🗄️ Data Analytics - Week 3 (SQL Data Analysis)

## 🚀 Overview

This repository contains my **Week 3 Data Analytics Project** completed as part of training at DecodeLabs.

In this project, I worked on:

* **SQL Data Analysis**
* Extracting insights from dataset using structured queries

---

## 📁 Files Included

* `Cleaned_Dataset.xlsx` → Clean dataset (from Week 1)
* `SQL Queries.sql` / Notebook → SQL queries used for analysis

---

## 🎯 Objective

The main goal of this project is to:

* Use SQL to query data
* Extract meaningful insights
* Perform aggregations and filtering

---

## 🛠 Tools & Technologies

* SQL (MySQL / SQLite / PostgreSQL)
* Excel (for dataset)
* Database tools (DB Browser / MySQL Workbench)

---

## 🧾 SQL Operations Performed

### 🔹 Basic Queries

* SELECT all data
* Count total records

### 🔹 Aggregations

* COUNT() → total orders
* SUM() → total sales
* AVG() → average order value

### 🔹 Grouping

* GROUP BY Product
* GROUP BY OrderStatus
* GROUP BY ReferralSource

### 🔹 Filtering

* WHERE clause (e.g., cancelled orders)

### 🔹 Sorting

* ORDER BY (highest to lowest values)

---

## 📊 Key Insights

* Total number of orders calculated using COUNT()
* Total sales and average order value derived using SUM() and AVG()
* Product-wise sales showed which products generate more revenue
* Referral source analysis identified top traffic sources
* Order status analysis highlighted high cancellations and returns

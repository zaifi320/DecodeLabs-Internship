The main goal of this project is to:

## 🛠 Tools & Technologies

## 📊 Key Analysis Performed

### 🔹 Basic Statistics

* Mean, Median, Count
* Data distribution analysis

### 🔹 Data Exploration

* Product-wise sales analysis
* Quantity analysis
* Payment method distribution
* Order status analysis
* Referral source analysis

### 🔹 Pivot Table Analysis (Excel)

* Sales by Product
* Quantity by Product
* Payment Method vs Product
* Order Status breakdown
* Referral Source performance

---

## 📈 Key Insights

* Laptop generated the highest revenue
* Online payment is the most preferred method
* High number of cancelled and returned orders
* Instagram is the top referral source
* Product demand varies across categories
